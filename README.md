Website with my projects, published to https://github.com/mterczynski/mterczynski.github.io

## Running the app locally

Install dependencies:

```bash
npm install
```

Start a local static server:

```bash
npm start
```

Then open http://localhost:8080 in your browser.

Alternative workflow: use `peakchen90.open-html-in-browser` in VS Code.

## Scripts

Deploy script - creates and pushes a commit in mterczynski.github.io

```
sh deploy.sh
```

Pipelines can be tracked at https://github.com/mterczynski/mterczynski.github.io/actions
